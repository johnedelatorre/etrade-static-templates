Getting started

Installing dependencies

  • npm install -g harp@0.15.2
  • npm install -g bower

Note there is an issue with the current versions of harp & node. Specifying harp v0.15.2 resolves the issue

Getting the project

  1. Clone the repo.
  2. Copy public/trading-conditional folder to its own diectory. You don't need anything else from the tearsheets repo.

Running the project

The source code uses harp, an easy to use static site generator. Harp compiles .ejs into .html and .scss into .css. It has layout and partial conventions that make it easy to reuse code.

Use these scripts from package.json to run and build the project.

  • npm start - will install your bower deps and start a harp server on port 8080. harp compiles .ejs and .scss as you work so no need to watch/compile anything.
  • npm run build - will compile your src to a dist folder and zip dist into a download.zip file.
